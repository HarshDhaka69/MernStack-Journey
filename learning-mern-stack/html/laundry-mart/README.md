# Laundry Mart

Laundry Mart is a simple HTML5 project that represents a basic laundry service website. It is designed to practice fundamental HTML concepts such as semantic tags, tables, forms, images, and lists.

## Features

- Welcome Header
- About Us Section
- Laundry Service Image
- Services List
- Price List Table
- Booking Form
- Contact Information Footer

## Technologies Used

- HTML5

## Project Structure

```
Laundry Mart/
│── index.html
│── README.md
```

## How to Run

Open `index.html` in any web browser.
