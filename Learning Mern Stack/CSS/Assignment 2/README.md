# CSS Units, Box Model & Fonts

This project demonstrates the use of CSS Units, Box Model, Fonts, and Gradient Backgrounds to create a simple responsive webpage.

## Features

- Responsive layout using `vw` and `vh`
- CSS Box Model (Margin, Padding, Border)
- Gradient button background
- Image display
- Custom button styling
- External CSS file

## Technologies Used

- HTML5
- CSS3
