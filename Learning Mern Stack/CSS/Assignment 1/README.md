# CSS Selectors

A beginner CSS project demonstrating the use of different CSS selectors to style HTML elements.

## Features

- Element Selector
- Class Selector
- ID Selector
- Direct Child Selector
- Descendant Selector
- Styled Links and Lists

## Technologies Used

- HTML5
- CSS3

## How to Run

Open `index.html` in any web browser.
