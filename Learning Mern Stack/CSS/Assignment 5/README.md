# Responsive Navigation Bar

A responsive navigation bar built using HTML and CSS without JavaScript. This project demonstrates responsive web design using Media Queries, a mobile-friendly navigation menu, and CSS pseudo-classes.

## Features

- Responsive navigation bar
- Desktop and mobile navigation layouts
- CSS-only hamburger menu
- Media Queries for responsive design
- Hero section layout
- External CSS styling

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
.

```

## How to Run

Open `index.html` in any modern web browser.
