# Restaurant Website

A CSS-based mini restaurant website built to practice modern webpage layouts and styling techniques. The project currently includes the desktop version, with tablet and mobile responsiveness planned for future updates.

## Features

- Desktop-first responsive layout
- Modern landing page design
- Navigation bar
- Hero section
- Menu section
- About section
- Contact section
- Footer Section
- CSS animations and hover effects
- Clean and organized UI

## Technologies Used

- HTML5
- CSS3

## Current Status

- ✅ Desktop version completed
- 🚧 Tablet responsiveness in progress
- 🚧 Mobile responsiveness in progress

- Add mobile responsive layout
