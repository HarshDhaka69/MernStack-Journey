# Harsh Task 6 — Login Page

A responsive login page built with pure HTML and CSS.

## Preview

Two-column layout with an image on the left and a login form on the right.

## Structure

```
Harsh_Task6/
├── index.html
├── style.css
└── README.md

```

## Features

- Two column layout using Flexbox
- Left panel with branding image
- Right panel with login form
- Email and password inputs
- Forgot password link
- Register redirect link

## How to Run

Open `index.html` in any browser


## Built With

- HTML5
- CSS3 (Flexbox)

## Layout

| Section | Width |
|---------|-------|
| Left panel (image) | 50% |
| Right panel (form) | 50% |

## Notes

- Spacing done without margin/padding (assignment constraint)
- Inputs styled with `background-color: #e1e3e7` and `border-radius: 20px`
- Button styled with `background-color: #0d6efd`
