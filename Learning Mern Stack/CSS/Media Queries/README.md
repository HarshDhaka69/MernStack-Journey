# Responsive Webpage using Media Queries

A responsive webpage built with HTML and CSS to practice Media Queries. The layout adapts to different screen sizes, providing a better user experience across desktop, tablet, and mobile devices.

## Features

- Responsive design using Media Queries
- Clean and structured layout
- HTML5 semantic structure
- External CSS styling

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
.
├── index.html
├── style.css
└── README.md
```

## How to Run

Open `index.html` in any modern web browser.
