# Laundry Service Hero Section

A simple landing page hero section built using HTML and CSS without using Flexbox. This project demonstrates the use of floats, viewport units, and the CSS box model to create a responsive layout.

## Features

- Responsive navigation bar
- Hero section with text and image
- Float-based layout
- Styled call-to-action button
- External CSS styling

## Technologies Used

- HTML5
- CSS3
